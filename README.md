# Gestión de Licitaciones - CSC Caballero

Sistema de gestión de licitaciones desarrollado en Laravel con un entorno contenedorizado en Docker utilizando Laravel Sail.

## Requisitos Previos
* Docker Desktop
* Git

## Arquitectura del Entorno (Docker Sail)
* **Application Server:** PHP 8.4
* **Database:** PostgreSQL

## Comandos Útiles
* **Encender entorno:** `./vendor/bin/sail up -d`
* **Apagar entorno:** `./vendor/bin/sail down`
